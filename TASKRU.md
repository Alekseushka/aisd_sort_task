# Сортировка по "удаленности" от идеала

Условие:
Дан массив целых чисел nums и целевое число target. Отсортируйте массив по следующим правилам:

    Числа сортируются по минимальной разнице между числом и любым элементом из target (если target - массив) или просто разнице с target (если target - число)

    Если два числа имеют одинаковую разницу, первым должно идти число, которое ближе к среднему значению всего массива

    Если и это одинаково, то первым идет меньшее число

Пример 1:
text

Ввод: nums = [1, 5, 9, 3, 7], target = 4
Вывод: [3, 5, 1, 7, 9]

Объяснение:

    Разницы: |3-4|=1, |5-4|=1, |1-4|=3, |7-4|=3, |9-4|=5

    3 и 5 имеют одинаковую разницу (1), но среднее массива = 5, |3-5|=2, |5-5|=0 → 5 ближе к среднему, поэтому 5 идет после 3

Пример 2:
text

Ввод: nums = [10, 2, 8, 15, 12], target = [5, 13]
Вывод: [8, 12, 10, 15, 2]

Объяснение:

    Минимальные разницы: min(|8-5|,|8-13|)=5, min(|12-5|,|12-13|)=1, min(|10-5|,|10-13|)=3, min(|15-5|,|15-13|)=2, min(|2-5|,|2-13|)=3

    Сортировка по разнице: 12(1), 15(2), 10(3), 2(3), 8(5)

    10 и 2 имеют одинаковую разницу (3), среднее = 9.4, |10-9.4|=0.6, |2-9.4|=7.4 → 10 ближе
