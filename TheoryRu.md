# hfls
## Основная концепция: Сортировка с несколькими критериями сравнения, где каждый следующий критерий применяется только при равенстве предыдущих.

## Ключевые элементы:

    Компаратор - объект, определяющий порядок элементов

    Приоритет критериев - строгая последовательность проверки условий

    Стабильность - сохранение относительного порядка равных элементов

## Алгоритм работы:

if (критерий1 ≠) → вернуть сравнение по критерию1
else if (критерий2 ≠) → вернуть сравнение по критерию2  
else → вернуть сравнение по критерию3

Сложность: O(n log n) время, O(n) память
